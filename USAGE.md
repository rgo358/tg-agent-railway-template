# 📱 Как использовать TG Agent

## Общая информация

**TG Agent** - это Telegram бот, который:
- 📚 Индексирует все твои сообщения за 3 месяца
- 🤖 Анализирует их через Groq AI (бесплатно)
- 💬 Отвечает на обычные вопросы на русском языке
- 🔍 Находит похожие сообщения из твоих чатов

**Все взаимодействие через "Сохраненные сообщения"** в Telegram!

---

## ✍️ Как писать агенту

### **Команды на переиндексацию:**

Агент поймет если ты напишешь:
```
обнови
индекс
обновить базу
переиндексируй
заново
загрузи данные
перезагрузи
```

**Пример:**
```
Пользователь: обнови
Агент: ⏳ Переиндексирую базу данных за последние 3 месяца...
(ждет, индексирует)
Агент: ✅ Индексация завершена!
       📊 Обработано:
       • 1250 сообщений
       • 15 чатов/каналов
```

### **Вопросы и анализ:**

Пишешь **обычный русский язык** - агент поймет:

| Что пишешь | Что сделает |
|-----------|-----------|
| `что про крипто?` | Поищет все про крипто в твоих чатах, потом спросит Groq что об этом думает |
| `какие новости за неделю?` | Найдет свежие сообщения, выделит самые интересные |
| `кто сказал про отпуск?` | Поищет упоминания отпуска, покажет контекст |
| `анализ графиков` | Поищет обсуждения графиков, сделает анализ |
| `что важного произошло?` | Выделит самые важные события из чатов |

**Пример взаимодействия:**
```
Пользователь: 📈 Что нового в крипте?
Агент: 🤖 Groq анализирует...
Агент: 🔍 На основе твоих чатов за 3 месяца:
       
       📢 Канал crypto_news: Bitcoin + 15%
       👥 Группа moneymakers: Давайте в Ethereum
       📢 Канал analiz_tech: Вероятна коррекция
       
       💡 Вывод: В крипто положительный тренд, расчеты на продолжение роста. Обсуждается переход в альткойны.
```

## 📊 Что разумеет агент

✅ **Работает:**
- Обычный русский язык (любой регистр)
- Вопросы и анализ
- Команды на переиндексацию
- Долгие вопросы
- Специальные символы и эмодзи

❌ **Не работает:**
- Голосовые сообщения
- Геопозиция
- Файлы и медиа
- Чужие боты в чате (только твои сообщения)

## ⚠️ Важные моменты

1. **Первый запуск медленный** - агент индексирует все сообщения (может занять 1-5 минут)
2. **Последующие анализы быстро** - ищет в уже готовой базе
3. **Бесплатный API** - Groq даёт бесплатно 30 запросов в минуту
4. **Только в личке** - агент слушает только "Сохраненные сообщения"
5. **Контекст за 3 месяца** - индексирует только последние 90 дней

## 🆘 Troubleshooting

### "Ничего не нашёл по запросу"
- Попробуй перефразировать вопрос
- Убедись что в чатах есть релевантные сообщения
- Запусти переиндексацию: напиши "обнови"

### "Ошибка анализа"
- Проверь что Groq API работает (доступен в https://console.groq.com)
- Попробуй запрос еще раз через 30 секунд
- Проверь Сохранённые сообщения - там могут быть подробные логи

### Агент не отвечает
- Проверь что агент запущен: `ps aux | grep tg_super`
- Перезапусти: `pkill -f tg_super_agent && python tg_super_agent.py`
- Проверь логи: `tail -100 /tmp/agent_groq.log`

---

## 💡 Советы

1. **Форматируй вопросы четко** - по одному вопросу в сообщении
2. **Используй контекст** - указывай период ("за неделю", "за день", "за месяц")
3. **Переиндексируй регулярно** - если в чатах новые важные сообщения
4. **Проверяй Telegram Saved** - результаты видны только там
