# 🔑 Как получить API ключи

Здесь пошаговые инструкции как получить все необходимые ключи и запустить агент.

---

## 1️⃣ SESSION_STRING (Telegram сессия)

### Что это?
Это строка, которая позволяет агенту подключаться от твоего имени к Telegram.

### Как получить?

#### Шаг 1: Получи API_ID и API_HASH (см. пункт 2 ниже)

#### Шаг 2: Запусти скрипт
```bash
python get_session.py
```

#### Шаг 3: Введи данные
```
Введи API_ID из my.telegram.org (число): 12345678
Введи API_HASH из my.telegram.org (строка): abc123def456...
```

#### Шаг 4: Авторизуйся в Telegram
Программа откроет окно авторизации:
```
Please enter your phone (or bot token): +380XXXXXXXXX
(введи свой номер телефона начиная с кода страны)

Please enter the code you received: 12345
(введи код подтверждения из Telegram)

Please enter your password: (if you have 2FA)
(если включена двухфакторная аутентификация)
```

#### Шаг 5: Скопируй результат
Программа выведет строку:
```
==================================================
✅ Твоя SESSION_STRING:
==================================================
1ApWapzMBuxQoToFu0oSNqQQQVDNAGfkN2cr-Wf5B7hj1hDOMc8dJVXc_...
==================================================
Используй эту строку в Railway!
==================================================
```

**Сохрани эту строку - она твой ключ доступа!**

## 2️⃣ API_ID и API_HASH

### Что это?
Это учетные данные для работы с Telegram API. Они нужны чтобы подключиться к Telegram серверам.

### Как получить?

#### Способ 1: Стандартный (рекомендуется)

1. Открой https://my.telegram.org/apps
2. Логин в свой аккаунт Telegram
3. Введи номер телефона и код подтверждения из смс

Ты увидишь говорка "Your Applications":

4. Нажми **"Create New Application"** (если первый раз) или используй существующее

5. Заполни форму:
   - **App name**: `MyAgent` (любое имя)
   - **Short name**: `myagent` (только буквы и цифры)
   - **URL**: `http://localhost` (не важно для нашего случая)
   - **Description**: `Personal Telegram AI Assistant` (описание)

6. После создания ты увидишь:
   ```
   api_id = 12345678
   api_hash = "abc123def456ghi789jkl012mnop..."
   ```

7. Сохрани эти значения

## 3️⃣ GROQ_API_KEY

### Что это?
Это ключ доступа к бесплатному Groq API, который предоставляет LLM модель (mixtral-8x7b). Groq - это очень быстрый и бесплатный AI сервис.

### Почему Groq?
- ✅ **Бесплатно** - 30 запросов в минуту без кредитов
- ✅ **Быстро** - ответ за 2-5 секунд
- ✅ **Мощно** - mixtral-8x7b лучше чем ChatGPT 3.5
- ✅ **Надежно** - стабильный сервис

### Как получить?

#### Шаг 1: Регистрация
1. Открой https://console.groq.com
2. Нажми **"Sign Up"** (если нет аккаунта)
3. Регистрируйся через Google, GitHub или email
4. Проверь email (может быть в спаме)

#### Шаг 2: Создание API ключа
1. Открой https://console.groq.com/keys
2. Нажми **"Create New API Key"**
3. Назови его: `telegram-agent` (любое имя)
4. Нажми **"Create"**

#### Шаг 3: Скопируй ключ
Ты увидишь строку начинающуюся с `gsk_`. 

**Сохрани этот ключ - он не покажется снова!**

Если потеряешь - создашь новый нажав "Create New API Key" еще раз.

## ❓ Частые вопросы

### Q: Нужна ли кредитная карта для Groq?
**A:** Нет! Groq полностью бесплатный для первых 30 запросов в минуту. Во вкладке https://console.groq.com вообще нет места для кредитных карт.

### Q: Нужна ли кредитная карта для Telegram API?
**A:** Нет! Telegram API абсолютно бесплатный.

### Q: Можно ли использовать старые ключи?
**A:** SESSION_STRING действителен пока ты не логаешься из другого места. API_ID/API_HASH никогда не меняются (это учетные данные приложения).

### Q: Что если SESSION_STRING истек?
**A:** Просто создай новую через `python get_session.py` еще раз.

### Q: Могу ли я использовать один API_ID для разных приложений?
**A:** Да, но Telegram рекомендует использовать разные для разных приложений.

### Q: Groq ограничивает 30 запросов в минуту - этого хватит?
**A:** Да! В обычном использовании ты пишешь несколько вопросов в минуту, максимум 10-15. 30 запросов достаточно.

### Q: Что если превышу лимит?
**A:** Groq просто вернет ошибку "Rate limit exceeded" (слишком много запросов). Подожди минуту и попробуй снова.
