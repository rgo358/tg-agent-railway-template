# 🤖 TG Agent — Анализатор Telegram сообщений с AI

**Личный ассистент, который анализирует все твои Telegram чаты через Groq AI (бесплатно)**

Агент собирает сообщения за **последние 30 дней**, индексирует их, и отвечает на **любые вопросы на русском языке** через "Сохраненные сообщения".

---

## ✨ Возможности

✅ **Индексация** - автоматически скачивает последние 30 дней из всех твоих групп/каналов  
✅ **Поиск** - находит похожие сообщения через векторный поиск (ChromaDB)  
✅ **Анализ AI** - анализирует найденные сообщения через Groq (mixtral-8x7b)  
✅ **Естественный язык** - понимает обычный русский язык, не нужны команды  
✅ **Telegram интеграция** - все через "Сохраненные сообщения"  
✅ **100% бесплатно** - Groq Cloud API предоставляет 30 запросов/минуту бесплатно  
✅ **Railway ready** - легко развертывается на Railway.app  

---

## 🚀 Быстрый старт

### 1️⃣ **Получи данные для подключения**

Тебе нужны 4 значения:

| Что | Где получить |
|-----|-------------|
| **SESSION_STRING** | Запусти локально: `python get_session.py` (авторизуйся в Telegram) |
| **API_ID** | https://my.telegram.org/apps → создай приложение |
| **API_HASH** | Там же, в приложении |
| **GROQ_API_KEY** | https://console.groq.com/keys → создай ключ |

### 2️⃣ **Локальный запуск**

```bash
# Установи зависимости
pip install -r requirements.txt

# Запусти агент
export SESSION_STRING="твой_ключ_здесь"
export API_ID="твой_id"
export API_HASH="твой_hash"
export GROQ_API_KEY="твой_groq_ключ"

python tg_super_agent.py
```

## 🆘 Troubleshooting

| Проблема | Решение |
|----------|---------|
| "Модель не найдена" | Переин install зависимостей: `pip install -r requirements.txt` |

| "Groq API ошибка" | Проверь ключ на https://console.groq.com |
| Агент не отвечает | `pkill -f tg_super && python tg_super_agent.py` |

## 🤝 Как получить KEY?

### SESSION_STRING (Telegram сессия)
```bash
python get_session.py
# Введи API_ID и API_HASH
# Авторизуйся в Telegram через смс
# Получишь строку SESSION_STRING
```

### API_ID и API_HASH
1. Открой https://my.telegram.org/apps
2. Логин + смс
3. Создай приложение (или используй существующее)
4. Скопируй API_ID и API_HASH

### GROQ_API_KEY
1. Открой https://console.groq.com/keys
2. Нажми **"Create API Key"**
3. Скопируй значение (начинается с `gsk_`)
