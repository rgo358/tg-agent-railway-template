# TG Agent — личный анализатор Telegram

Агент видит все твои группы и каналы, индексирует сообщения за 3 месяца и отвечает на простые вопросы.

## Деплой одним кликом

[![Deploy to Railway](https://railway.app/button.svg)](https://railway.app/new/template?template=https%3A%2F%2Fgithub.com%2Frgo358%2Ftg-agent-railway-template)

После клика Railway сам попросит ввести:
- SESSION_STRING
- API_ID
- API_HASH
- XAI_API_KEY
